SARS-CoV-2 3C-like protease (a.k.a. NSP5, Mpro, main protease) (3CL) gene 3CL:p.L50F;E166V literature reference collection

Pharmaceutical effectiveness

  • In a escape mutation generation experiment using protease inhibitor nirmatrelvir, one of two independent experiments generated this mutation combination. It provides 175x resistance to nirmatrelvir as measured by EC50 in Vero E6 cells.
    The significance of these putative resistance substitutions was confirmed by reverse genetics using a bacterial artificial chromosome (BAC) clone, which displayed ~80x resistance.
    Remdesivir effectiveness was not significantly affected by this mutation combination, suggesting that ciombined treatment would be less likely to induce resistance.
    (Zhou et al. (2022))
